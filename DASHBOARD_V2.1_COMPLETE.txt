NEW COMPONENT: EnhancedAnalytics (src/components/EnhancedAnalytics.tsx)

This powerful analytics component provides:

✨ SYSTEM OVERVIEW PANEL
   • Total event count (real-time)
   • Current threat level (Low/Medium/High/Critical)
   • Number of active platforms/sources
   • Trending topic count

✨ EVENT DISTRIBUTION ANALYSIS
   • Events per platform (Twitter, Reddit, YouTube, News, etc.)
   • Visual bar charts with percentages
   • Helps identify primary intelligence channels
   • Dynamic layout based on active sources

✨ CONFIDENCE LEVELS METRICS
   • High Confidence (>70%): Verified sources
   • Medium Confidence (40-70%): Moderate validation
   • Low Confidence (<40%): Preliminary data
   • Visual progress bars for each level
   • Color-coded indicators (Red, Yellow, Green)

✨ GEOGRAPHIC SPREAD VISUALIZATION
   • North America (25-50°N, -125--65°E)
   • Europe (35-70°N, -10-40°E)
   • Asia (0-50°N, 60-150°E)
   • Other Regions (remaining areas)
   • Proportional regional distribution
   • Color-coded by region (Blue, Green, Orange, Purple)

✨ TRENDING TOPICS EXTRACTION
   • Auto-extracts keywords from event text
   • Filters out common words
   • Top 10 trending topics displayed
   • Tag-style visualization
   • Real-time updates as new events arrive

✨ AUTOMATED THREAT ASSESSMENT
   • Calculates threat level based on high-confidence events:
     - Low: < 5 events with confidence > 0.8
     - Medium: 5-10 events with confidence > 0.8
     - High: > 10 events with confidence > 0.8
     - Critical: (configurable threshold)
   • Detects threat factors:
     - Violence reports
     - Protest activity
     - Emergency situations
     - Critical incidents
   • Color-coded backgrounds (Green/Yellow/Orange/Red)

HOW IT WORKS
================================================================================

DATA FLOW:

WebSocket Connection (ws://localhost:8000/ws/intelligence)
         ↓
    Sample Data Generation (2-second intervals)
         ↓
useWebSocket Hook
         ↓
addEvent() → Zustand Store
         ↓
Events Array (Max 100, auto-trimmed)
         ↓
┌─────────────────────────────────────┐
│   Real-time Computation             │
│   • Clustering (DBSCAN)             │
│   • Analytics (EnhancedAnalytics)   │
└─────────────────────────────────────┘
         ↓
Dashboard OR Analytics Component
         ↓
Real-time Rendering (60 FPS)

ANALYTICS CALCULATION:

Events → Platform Distribution (group by source)
     → Confidence Analysis (filter by 0.7, 0.4 thresholds)
     → Geographic Analysis (filter by lat/lng ranges)
     → Text Analysis (extract keywords from text field)
     → Threat Assessment (count high-confidence + threat factors)
     → Display Results (real-time render)

TIME COMPLEXITY: O(n) for most operations, n = number of events
