🎯 CORE COMPONENTS EXPLAINED
───────────────────────────────────────────────────────────────────────────────

1. ENHANCED GLOBE (EnhancedGlobe.tsx)
   ─────────────────────────────────────
   The 3D world map you requested.
   
   What it does:
   • Visualizes events on a rotating 3D Earth
   • Uses real geographic coordinates (latitude/longitude)
   • Colors events by severity (RED/ORANGE/YELLOW/GREEN)
   • Shows event counts
   • Interactive: hover to focus on regions
   • Auto-rotates with smooth animations
   
   Visual:
   ```
   ┌──────────────────────────────┐
   │     3D ROTATING GLOBE        │
   │        ╭─────────────╮       │
   │       ╱   🔴🟡🟢     ╲      │
   │      │  🌍 Events   │      │ ← Events plotted
   │       ╲              ╱       │
   │        ╰─────────────╯       │
   │   [Legend]  345 Points      │
   └──────────────────────────────┘
   ```

2. HEATMAP GLOBE (HeatmapGlobe.tsx)
   ────────────────────────────────
   Alternative 2D heatmap visualization.
   
   What it does:
   • Shows signal density by region
   • Aggregates events by location
   • Uses color intensity (lighter = fewer, darker = more)
   • Hex-based binning for efficiency
   
   Toggle: Click "3D" or "HEATMAP" button to switch

3. SOURCE PANEL (SourcePanel.tsx)
   ──────────────────────────────
   The information sources you wanted.
   
   Shows 6 monitored sources:
   • Twitter     (real-time social signals)
   • Reddit      (community discussions)
   • YouTube     (video content)
   • News        (news outlets)
   • Instagram   (image intelligence)
   • LinkedIn    (professional network)
   
   For each:
   ✅ Live count (e.g., "456")
   ✅ Percentage of total (e.g., "36.5%")
   ✅ Visual progress bar
   ✅ Total events counter
   
   Updates in REAL-TIME as new events arrive!

4. LIVE FEED (LiveFeed.tsx)
   ───────────────────────
   Real-time intelligence stream.
   
   Shows: Latest 10 events with:
   • Event content
   • Source (Twitter, Reddit, etc.)
   • Severity (with color)
   • Location (country/coordinates)
   • Timestamp
   • Full event details

5. THREAT LEVEL (ThreatBar.tsx)
   ───────────────────────────
   Dynamic threat indicator.
   
   Shows:
   • 0-100% threat scale
   • Color zones:
     🟢 GREEN (0-25%) - Normal
     🟡 YELLOW (25-50%) - Elevated
     🟠 ORANGE (50-75%) - High Alert
     🔴 RED (75-100%) - Critical
   • Updates in real-time
