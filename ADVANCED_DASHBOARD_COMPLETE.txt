BACKEND INTEGRATION
================================================================================

To connect to real data (replace demo mode):

1. Ensure your WebSocket server runs on:
   ws://localhost:8000/ws/intelligence

2. Send events in this format:
   {
     "events": [
       {
         "id": "event-uuid",
         "timestamp": 1704067200000,
         "source": "Twitter",
         "severity": "critical",
         "content": "Event description here",
         "platform": "Twitter",
         "confidence": 0.95,
         "location": {
           "lat": 40.7128,
           "lng": -74.0060,
           "city": "New York",
           "country": "USA"
         }
       }
     ]
   }

3. Dashboard automatically connects and displays data

4. Demo mode disables automatically when real data arrives

TROUBLESHOOTING
================================================================================

Problem: "Module not found" error
Solution: Run: npm install

Problem: No data appears on dashboard
Solution: Wait 2 seconds for auto-generated sample data

Problem: Globe won't rotate
Solution: Click and drag on globe to rotate manually

Problem: Slow performance / freezing
Solution: Close other applications; check memory in DevTools

Problem: npm install fails
Solution: Update Node.js to 16+; clear npm cache

Problem: WebGL error
Solution: Update GPU drivers; use modern browser
